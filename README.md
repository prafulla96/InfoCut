# InfoCut

InfoCut is a Razor Pages web application built with ASP.NET Core (.NET 9, C# 13) for delivering daily updates, news, and user interaction features.

## Features

- **User Authentication:** Sign up and sign in functionality with session management.
- **Profile Management:** View user profile after authentication.
- **Content Pages:** Includes sections for Daily Updates, Economy, Top 10, Sports, Tech, Privacy, and Terms.
- **Contact & About:** Dedicated pages for contact information and about the project.

## Getting Started

### Prerequisites

- [.NET 9 SDK](https://dotnet.microsoft.com/download/dotnet/9.0)
- Visual Studio 2022 or later
- SQL Server (or compatible database for Entity Framework Core)
